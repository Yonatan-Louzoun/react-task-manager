# אפליקציית מנהל משימות - פרויקט מסכם

## סקירה כללית
אפליקציית React מלאה לניהול משימות המאפשרת הוספה, עריכה, מחיקה וסינון משימות. הפרויקט מדגים שימוש ב-React 19, ניהול State, Props, ושמירת נתונים ב-LocalStorage.

## הוראות הרצה מקומית
1. התקנת ספריות: `npm install`
2. הרצת פיתוח: `npm run dev`

## רשימת קומפוננטות ואחריותן
* **App**: ניהול ה-State המרכזי, לוגיקת הסינון וסנכרון עם LocalStorage.
* **TaskInput**: רכיב לטיפול בקלט המשתמש והוספת משימות חדשות.
* **TaskList**: ניהול תצוגת אוסף המשימות ורינדור הרשימה המסוננת.
* **TaskItem**: הצגת משימה בודדת, ניהול מצב עריכה (Edit Mode) ושינוי סטטוס.
* **FilterBar**: תצוגת כפתורי הסינון ומונה המשימות הפעילות שנותרו.

## תכונות ממומשות
* שמירה וטעינה מ-LocalStorage (המשימות נשמרות ברענון דף).
* סינון לפי: הכל, פעילות, והושלמו.
* עריכת משימות קיימות.
* מונה משימות פעילות בזמן אמת.
* בונוס: כפתור "נקה הושלמו" למחיקת כל המשימות שבוצעו.
